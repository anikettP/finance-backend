Build a Finance Dashboard Backend using Node.js, Express, and MongoDB.

Requirements:
- JWT Authentication (login/register)
- Role-based access (viewer, analyst, admin)
- CRUD for financial records
- Dashboard APIs:
    - total income
    - total expense
    - net balance
    - category breakdown
- Filtering, pagination, search
- Validation and error handling

Bonus:
- Swagger docs
- Rate limiting
- Soft delete
